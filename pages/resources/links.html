Resource Navigation

资源导航

按学习场景分类整理的常用网站和工具，涵盖语法文档、学习平台、开发工具和设计灵感，方便日常开发时快速查阅。

语法文档

写代码时最常翻的参考站

HTML / CSS / JS MDN Web Docs

Mozilla 维护的前端技术文档，语法说明和示例都很完整，遇到 API 不确定时第一个查的地方。

 兼容性 Can I use

输入 CSS 属性或 JS API 就能看到各浏览器的支持情况，答辩时解释兼容性考虑很好用。

 快速查阅 W3School 中文

每个标签和属性都有在线编辑器可以直接试，适合快速确认某个写法的效果。

学习平台

课堂外的补充学习资源

基础复盘 菜鸟教程

适合快速回顾基础知识点，每个章节都有简洁的代码示例和在线运行环境。

 视频教程 哔哩哔哩

前端学习视频最集中的平台，从零基础入门到框架实战都能找到对应课程。

 技术社区 稀土掘金

国内前端开发者活跃度最高的社区之一，实战经验和最新技术动态都能看到。

开发工具

日常开发中离不开的工具

代码托管 GitHub

全球最大的代码托管平台，可以观察优秀项目如何组织代码结构和文档。

 编辑器 VS Code

微软出品的轻量级代码编辑器，插件生态丰富，前端开发的主力工具。

 在线编码 CodePen

在线编写 HTML/CSS/JS 并实时预览效果，非常适合快速验证某个想法或调试样式。

设计灵感

页面设计时的参考来源

UI 设计 Dribbble

设计师社区，配色方案、卡片布局、页面风格都能在这里找到灵感。

 配色方案 Color Hunt

按风格分类的配色卡合集，选颜色时不用自己从零调配，直接挑选组合即可。

分类索引

快速跳转

  • 学习平台 - 菜鸟教程 / B站 / 掘金
  • 开发工具 - GitHub / VS Code / CodePen
  • 设计灵感 - Dribbble / Color Hunt

使用建议

怎么用这些资源

遇到具体语法问题先查 MDN，需要看整体用法时去菜鸟教程。

样式没灵感就翻 Dribbble，颜色拿不准就去 Color Hunt 找现成方案。

写完代码想快速验证，CodePen 比本地起服务更方便。
